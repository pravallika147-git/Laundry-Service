# Laundry Service Web Application

This project is a simple Laundry Service web application hero section built using HTML and CSS.

## Features
- Navigation bar with Logo, Home, Services, About Us, and Contact Us
- Hero section with heading, description, and booking button
- Laundry service image
- Responsive layout using viewport units

## Technologies Used
- HTML
- CSS

## Project Objective
The objective of this project is to design and implement the Hero Section of a laundry service web application that improves the user experience.
